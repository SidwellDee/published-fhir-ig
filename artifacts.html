Eswatini Procedure Profile

This resource is designed to record details of current and historical procedures performed on a patient. This may include the start and stop dates of treatment like IPT.
Eswatini Specimen Profile

The Eswatini Specimen Profile is based on the FHIR Specimen resource. This profile sets minimum expectation for Specimen resource to record information about substances associated with a patient or laboratory request being tested. This profile specifies which core elements shall be present and how these elements are used.
Eswatini Vital Signs Profile

This profile is based on the base FHIR Vital Signs Profile and defines additional constraints on the Observation resource to represent vital signs observations specifically for Eswatini.
