Resource Profile: Eswatini Specimen Profile

Official URL: http://172.209.216.154:3447/fhir/StructureDefinition/SzLabSpecimen Version: 0.1.0
Active as of 2025-07-16 Computable Name: EswatiniSpecimenProfile

The Eswatini Specimen Profile is based on the FHIR Specimen resource. This profile sets minimum expectation for Specimen resource to record information about substances associated with a patient or laboratory request being tested. This profile specifies which core elements shall be present and how these elements are used.

Usages:

  • Refer to this Profile: Eswatini Lab Result Profile and Eswatini Lab Test Profile
  • Examples for this Profile: Specimen/SampleSzMalariaSpecimen and Specimen/SampleSzSpecimen
